## Directories to Submit

### 1. LiveBusiness.ca (Quebec)
- **URL:** https://livebusiness.ca/submit?c=89
- **Type:** Free, no registration required
- **Steps:**
  1. Navigate to the URL above
  2. Fill in business name: Smart Hotline
  3. Enter URL: https://www.smart-hotline.com
  4. Enter email: direction@smart-hotline.com
  5. Enter description (use French version above)
  6. Select category: Business Services
  7. Submit form
- **Expected:** Manual review, listing within 7-14 days

### 2. CanadianBusinessDirectory.ca
- **URL:** https://www.canadianbusinessdirectory.ca/submit.php
- **Type:** Free, requires registration
- **Steps:**
  1. Navigate to the URL above
  2. Fill registration form with business details
  3. Copy verification number shown on page
  4. Enter all required fields
  5. Select province: Quebec
  6. Submit form
- **Expected:** Email confirmation, listing within 14-30 days

### 3. TheBusiness.ca
