## Design Guidelines

### Visual Direction
- **Color Scheme:** Professional blue (#0066cc) and white with gray accents
- **Typography:** Clean, modern sans-serif fonts (system fonts for reliability)
- **Layout:** Clean, organized with good use of white space
- **Images:** Use existing images from /images/ directory
- **Responsive:** Mobile-first design with breakpoints for tablet and desktop

### Navigation Structure
- Main navigation with dropdown for Solutions
- Language switcher (FR/EN)
- Working links to all pages
- Mobile hamburger menu

### Key Features
- Hero section with call-to-action on homepage
- Service cards with icons
- Contact information in footer
- Professional, trustworthy appearance
- Fast loading times
- SEO-friendly structure

## Testing Requirements

Each page must:
- Load without errors
- Display correctly on mobile, tablet, and desktop
- Have working navigation links
- Load all images
- Function properly with CSS and JavaScript
- Be accessible (semantic HTML, alt text for images)
