## 📐 Video Specifications

| Setting | Recommended Value |
|---------|-------------------|
| **Resolution** | 1920x1080 (16:9) or 1080x1920 (9:16) |
| **Duration** | 15-30 seconds |
| **Format** | MP4 (H.264) |
| **Frame Rate** | 30fps |
| **File Size** | Under 10MB for web optimization |

**Aspect Ratio by Use:**
- Website hero sections: 16:9 (landscape)
- Mobile-first sections: 9:16 (portrait)
- Testimonials: 1:1 (square) or 16:9

## 📝 Step-by-Step Process

### Using Kling AI (Best for Testimonials)

1. **Create Account**
   - Go to [klingai.com](https://klingai.com)
   - Sign up with Google or email
   - Verify your account

2. **Generate Video**
   - Click "Create" or "Text to Video"
   - Select aspect ratio: 16:9
   - Set duration: 15-30 seconds
   - Paste your prompt (from above)
   - Click "Generate"
   - Wait 2-5 minutes

3. **Download**
   - Preview the generated video
   - Click "Download" (MP4 format)
   - Save with correct filename (e.g., `testimonial_fr.mp4`)

4. **Credits**
   - Free tier: ~66 videos/month
   - Each video uses 4-6 credits

---

### Using RunwayML (Best for Brand/Agent Videos)

1. **Create Account**
   - Go to [runwayml.com](https://runwayml.com)
   - Sign up (free tier available)
   - You get 125 credits/month

2. **Generate Video**
   - Go to "Gen-2" or "Gen-3 Alpha"
   - Select "Text to Video"
   - Choose resolution: 1080p
   - Paste your prompt
   - Set duration (up to 10s free, longer uses more credits)
   - Click "Generate"

3. **Refine if Needed**
   - Use "Motion Brush" to direct movement
   - Regenerate if unsatisfied
   - Download when happy

4. **Download**
   - Click the download icon
   - MP4 format, ready for web

---

### Using Canva (For Adding Logos & Text)

1. **Create Account**
   - Go to [canva.com](https://canva.com)
   - Sign up (free forever tier)

2. **Add Branding to Video**
   - Upload your AI-generated video
   - Add Smart Hotline logo (from `/public/images/logo.png`)
   - Add text overlays with brand colors (#1e40af blue, #ffffff white)
   - Add transitions if combining clips

3. **Export**
   - Click "Share" → "Download"
   - Format: MP4
   - Quality: 1080p
   - Download to your computer

---

### Using Pika Labs (Alternative)

1. **Create Account**
   - Go to [pika.art](https://pika.art)
   - Sign in with Discord or Google

2. **Generate**
   - Type `/create` or use web interface
   - Enter prompt
   - Select aspect ratio
   - Click generate

3. **Download**
   - Right-click → Save video
   - Rename appropriately

## 💾 Saving & Integrating Videos

### Step 1: Save Videos

Place all generated videos in:
```
/root/projects/smart-hotline-nextjs/public/videos/
```

**Required filenames:**
```
public/videos/
├── brand_fr.mp4        # French brand video
├── brand_en.mp4        # English brand video
├── agent_fr.mp4        # French agent video
├── agent_en.mp4        # English agent video
├── testimonial_fr.mp4  # French testimonial
├── testimonial_en.mp4  # English testimonial
└── VIDEO_GENERATION_INSTRUCTIONS.md  # This file
```

### Step 2: Optimize Videos (Optional but Recommended)

Before uploading, compress videos for web:

```bash
# Install ffmpeg if not available
# sudo apt install ffmpeg

# Compress each video (targets ~5MB per video)
cd /root/projects/smart-hotline-nextjs/public/videos

for file in brand_fr brand_en agent_fr agent_en testimonial_fr testimonial_en; do
  ffmpeg -i "${file}.mp4" -vcodec h264 -crf 28 -preset slow "${file}_optimized.mp4"
  mv "${file}_optimized.mp4" "${file}.mp4"
done
```

### Step 3: Update Website Components

The videos will be referenced in these components:

1. **Homepage Hero** (`app/[lang]/page.tsx`)
   - Uses `brand_fr.mp4` or `brand_en.mp4` based on language

2. **Agents Page** (`app/[lang]/agents/page.tsx`)
   - Uses `agent_fr.mp4` or `agent_en.mp4`

3. **Testimonials Section** (`components/Testimonials.tsx`)
   - Uses `testimonial_fr.mp4` or `testimonial_en.mp4`

### Step 4: Build & Deploy

```bash
cd /root/projects/smart-hotline-nextjs

# Build the site
rm -rf out && npm run build

# Verify videos are included
ls -la out/videos/

# Commit and push
git add public/videos/
git commit -m "Add marketing videos for FR and EN"
git push origin main

# Deploy
gh workflow run "Deploy to GitHub Pages" --repo boubahkarim-png/smart-hotline-last
```

## 🎨 Video Component Usage

### Basic Video Component

```tsx
// Example usage in React component
<video
  src="/videos/brand_fr.mp4"
  autoPlay
  loop
  muted
  playsInline
  className="w-full h-auto rounded-lg shadow-xl"
/>
```

### Language-Aware Video

```tsx
// Example: Auto-select video based on language
'use client';

export function BrandVideo({ lang }: { lang: 'fr' | 'en' }) {
  const videoSrc = lang === 'fr' 
    ? '/videos/brand_fr.mp4' 
    : '/videos/brand_en.mp4';

  return (
    <video
      src={videoSrc}
      autoPlay
      loop
      muted
      playsInline
      className="w-full h-auto"
    />
  );
}
```
