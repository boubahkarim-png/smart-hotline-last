# CRM Integration

## Overview
Smart Hotline provides full CRM integration services that connect call center operations with your existing business systems. Every call, message, and interaction is automatically logged and synced in real-time.

## Capabilities
- Salesforce integration with auto-logging
- HubSpot CRM sync for contacts and deals
- Custom API integration for proprietary systems
- Zoho CRM, Pipedrive, and Microsoft Dynamics support
- Real-time call data sync (caller ID, duration, transcription)
- Automated contact creation from incoming calls
- Lead scoring based on call interactions
- Custom webhook delivery for any endpoint
- ViciDial and Asterisk PBX integration
- Data export in CSV, JSON, and XML formats
